What is this domain used for?

Random ply.gg subdomains (such as random-domain.at.ply.gg) are assigned to the users / customers of playit.gg. playit.gg is a service that provides an alternative to port forwarding (primarily for game servers like Minecraft). These subdomains point to the playit network however the hosted content is provided by our user's computer / server.
